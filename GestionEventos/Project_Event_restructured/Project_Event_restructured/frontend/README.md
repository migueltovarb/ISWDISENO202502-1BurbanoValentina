# Frontend (Vite + React + Tailwind)

**UI en Español.** Conecta con el backend mediante `VITE_API_URL` (por defecto `http://localhost:8000`).

## Scripts

```bash
pnpm i # o npm i / yarn
pnpm dev
pnpm build
pnpm preview
```

## Configuración
Crear `.env` copiando de `.env.example` y ajustar `VITE_API_URL`.
